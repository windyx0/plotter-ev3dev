# Plott3r 🤖🖍️

Plott3r — это аппаратно-программный комплекс для создания рисунков с помощью LEGO Mindstorms EV3. Вы загружаете любую картинку через удобный веб-интерфейс, алгоритмы компьютера обрабатывают её (находят контуры, оптимизируют пути), генерируют G-code и отправляют напрямую на робота EV3 по Wi-Fi для отрисовки!

## 🌟 Основные фичи
- **Веб-интерфейс** — красивый и современный интерфейс для загрузки и обработки изображений прямо в браузере.
- **Поддержка любых устройств** — сервер можно запустить на Windows, Linux, macOS и даже на Android-телефоне (через Termux).
- **Разные режимы** — автоматическое превращение фото в линии (через фильтры Canny) или полностью ручное рисование собственных контуров.
- **Машинное зрение и оптимизация** — используются OpenCV и алгоритмы Рамера-Дугласа-Пекера (RDP) для идеального баланса детализации и скорости печати.
- **Калибровка и управление** — возможность двигать каретку, менять агрессивность дуг и парковать робота прямо с EV3 или ПК.

## 🛠️ Установка сервера (Компьютер / Смартфон)

Вся тяжелая обработка изображений происходит на сервере. Для удобства мы подготовили установочные скрипты для разных систем.

**1. Склонируйте репозиторий или скачайте файлы проекта.**
Откройте терминал в папке проекта (`plott3r_project`).

**2. Запустите скрипт установки для вашей ОС:**

- **Для Linux (Ubuntu, Debian и т.д.):**
  ```bash
  bash install/install_linux.sh
  ```
- **Для macOS:**
  ```bash
  bash install/install_macos.sh
  ```
- **Для Android (Termux):**
  ```bash
  bash install/install_termux.sh
  ```
  *(Скрипт сам подтянет тяжелые пакеты OpenCV и NumPy из специальных репозиториев, чтобы не нагружать телефон компиляцией).*

## 🚀 Запуск и использование

### Шаг 1. Запуск сервера
После установки вы можете запустить веб-интерфейс одним из стартовых скриптов:
- На Linux: `bash start_linux.sh`
- На macOS: `bash start_macos.sh`
- На Termux: `bash start_termux.sh`

После запуска в терминале появится адрес вида: `http://192.168.1.xxx:5000`. Откройте эту ссылку в браузере.

### Шаг 2. Подготовка EV3
1. Подключите ваш EV3 к той же Wi-Fi сети, что и компьютер/смартфон.
2. Скопируйте папку `ev3_side` на блок EV3 (например, через VS Code с расширением ev3dev).
3. На EV3 запустите скрипт `main_menu.py`.
4. В меню выберите `Print From Server` (робот выведет свой IP на экран).

### Шаг 3. Печать!
1. В веб-интерфейсе выберите картинку и настройте ползунки под желаемый результат.
2. Нажмите **"Обработать"** (Process) — справа появится предпросмотр результата.
3. Перейдите во вкладку **"Отправка на EV3"**.
4. Введите IP-адрес, который высветился на экране робота, и нажмите **"Отправить"**.
5. Наблюдайте, как Plott3r творит магию! 🎨

## 🔧 Структура проекта
- `computer_side/` — исходники Python-бэкенда (на Flask) и Frontend (HTML/JS/CSS).
- `ev3_side/` — код, который работает непосредственно на EV3 (управление моторами, парсинг G-code).
- `install/` — скрипты для автоматической установки зависимостей.
- `start_*.sh` — скрипты для быстрого старта веб-сервера.

## ⚙️ Меню на EV3
- **Continue Unleashed:** Продолжить печать, если она была прервана или отменена (прогресс сохраняется).
- **Print From Server:** Режим ожидания G-code команд по Wi-Fi.
- **Select File:** Печать G-code с локального файла на самом EV3.
- **Calibrate Menu:** 
  - `Full Calibrate` — откалибровать X и парковку бумаги.
  - `Eject Paper` — выплюнуть бумагу.
  - `Calibrate X` — откалибровать каретку ручки.
  - `Pen Test` — поднять/опустить ручку (Enter), выход (Влево/Вправо).

> 💡 **Совет:** Во время печати вы в любой момент можете нажать кнопки `Влево`, `Вправо` или `Назад` на EV3, чтобы отменить печать. Прогресс сохранится автоматически!
